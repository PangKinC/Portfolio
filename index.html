About Me


Hi my name is Kin, and I'm a C# and ASP.NET Developer from London.

For a long time now I've always had a keen interest and passion for anything IT related. Which only continued to grow to the point that I dedicated my studies entirely to it throughout both college and university, taking the BTEC National Diploma in IT course and Computer Science with Games Techonology degree respectively.

Starting from college education onwards, I've gained knowledge of the various different subjects and factors in IT ranging from Human Computer Interface (HCI), e-Commerce, Networking and much more.

Word Guessing Game

Overview


For this project, I created a simple word guessing game very similar to the popular game: Hangman. The objective of the game is to try get the highest score possible by the end after going through a set list of words.

In order to do this, the user should aim to not get any wrong guesses with incorrect letters to maintain a multiplier.
They should also should try guess the word as quick as possible for a higher score, and finally there's also one more multiplier for getting consecutive correct word guesses.

Restaurant Locator

Overview


For my second project, this time I attempted to use ASP.NET with the MVC (Model View Controller) design pattern.
I used in conjuction with this a database generated in SQL using SQL Management Studio, the project basically stores a list of different restaurants inside a table specifically tailored towards it.

The main feature of this project is it has a action which by taking in the latitude and longitude of a address, a user can first search for his address location and if it's found would then move to a page showing a Google Map with every restaurant located nearby said address. The amount of restaurant on the page is of course the same ones we stored in the database, and henceforth can be expanded much more in the future.
